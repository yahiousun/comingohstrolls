Overview 概览

What is it?

jQuery comingohstrolls plugin is a simple, no need extra markup and is used to smooth scrolling on the current page through the power and flexibility of jQuery´s selector.

jQuery comingohstrolls 插件是一个简单的，不需要额外标记的由jQuery驱动的平滑滚动插件。

Features 功能

  • 1. Smooth scrolling
  • 2. Response menu
  • 1. 平滑滚动
  • 2. 响应式菜单

How to use 如何使用

STEP-1

include these two javascript files in your header.

在你的header引用下面两个js文件.

<script type="text/javascript" src="[path]/jquery.min.js"></script>
<script type="text/javascript" src="[path]/jquery.comingohstrolls.min.js"></script>

STEP-2

HTML markup likes this:

使用如下HTML

<a href="#yourTargetId">your content</a>

Or:

或者

<ul id="#yourId">
	<li><a href="#yourTargetId1">your content</a></li>
	<li><a href="#yourTargetId2">your content</a></li>
	<li><a href="#yourTargetId3">your content</a></li>
	<li><a href="#yourTargetId4">your content</a></li>
	...
</ul>
<div id="yourTargetId1">your content</a>
<div id="yourTargetId2">your content</a>
<div id="yourTargetId3">your content</a>
<div id="yourTargetId4">your content</a>
...

STEP-3

Activate plugin

激活插件

<script>
$(document).ready(function(){
	$('Your Selector').comingohstrolls({
		response: true
	});
})
</script>
